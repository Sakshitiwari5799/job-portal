# job-portal
A full-stack job portal web application built using the MERN stack, designed to connect job seekers with recruiters through a modern, secure, and scalable platform. The application supports role-based access, allowing recruiters to post and manage jobs while candidates can search, apply, and track applications efficiently.

The project focuses on real-world functionality, clean architecture, and a responsive user experience, making it suitable for production-level deployment.
